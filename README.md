# Русификатор для игры Big Pharma для 1.08.12
Это последняя версия на осень 2025 года.
Изменения:
* исправлена часть орфографических ошибок;
* переведена часть названий лекарств.

## Установка русификатора
1. Скачать репозиторий (Code > Download ZIP).
2. Скопировать папку Big Pharma в место хранения папки игры.
3. Выбрать язык в списке языков.
